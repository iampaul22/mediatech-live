=== BuddyPress Analytics Lite by Dessky ===
Version: 1.0.2

== Description ==

Analytics plugin for BuddyPress. This is lite version of the premium BuddyPress Analytics plugin.

== Installation ==

1. Upload the buddypress-analytics-by-dessky folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'wp-admin > Settings > BuddyPress Analytics' page
